函数式编程

高级函数

变量可以指向函数

函数名也可以指向变量

一个函数可以作为参数传入另一个函数

map/reduce

map

对序列中的每个元素执行相同的操作
map()接受两种参数

  • Iterable
  • 函数名

reduce

对序列中的元素执行累加操作

filter

用来过滤序列
接受一个函数，作用于每一个元素，返回值是True保留该元素

sorted

还可以接受一个key函数来实现自定义的排序

匿名函数

只能有一个表达式，不能写return
返回值就是该表达式的结果

关键字lambda表示匿名函数，冒号前面的x表示函数参数
