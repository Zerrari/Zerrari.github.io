day4 in python

高级特性

切片

通过切片可以取出list和tuple或是字符串中的部分元素
索引从0开始
倒数第一个元素的索引为-1

固定步长切片

字符串

迭代

对dict使用迭代
默认迭代dict中的key

如果要迭代value
for value in d.values():

迭代字符串
for char in 'abcd':

列表生成式

if…else…

生成器

构造一个生成器

next(g)获取下一个元素
运用迭代来获得元素

迭代器

Iterable

Iterable可迭代对象
使用isinstance()判断是否为可迭代对象
isinstance('abc',Iterable)

Iterator

可以通过next()函数调用的迭代对象为迭代器Iterator
可以通过iter()函数获得Iterator

函数式编程

高级函数

变量可以指向函数

函数名也可以指向变量

一个函数可以作为参数传入另一个函数

map/reduce

map

对序列中的每个元素执行相同的操作
map()接受两种参数

  • Iterable
  • 函数名

reduce

对序列中的元素执行累加操作

filter

用来过滤序列
接受一个函数，作用于每一个元素，返回值是True保留该元素

sorted

还可以接受一个key函数来实现自定义的排序

匿名函数

只能有一个表达式，不能写return
返回值就是该表达式的结果

关键字lambda表示匿名函数，冒号前面的x表示函数参数
