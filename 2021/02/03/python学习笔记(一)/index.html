Posted on 2021-02-03 In Python

Day1 in python

脚本式编程

在命令行中运行python程序
python 1.py

运行python脚本
chmod +x 1.py
./1.py

chmod是一个用来改变文件权限的命令，*+x*为文件增加可执行权限

标识符

python中的标识符可以包含英文，数字和下划线，但不能以数字开头

行和缩进

python中的行的结束以新行的开始为标志
可以使用斜杠\来分隔多行
语句中包含 [], {} 或 () 括号就不需要使用多行连接符

print and input

print("hello","world")
('hello','world')

name=input()
print(name)

格式化输出
print("%d %d" % (1,3))
1 3

运算

/结果一定为浮点数
//结果为整数
%结果为余数

list

name=['Jordan','Mike','John']

方法

  • append
  • pop
  • insert

    索引

    python允许负的索引,从-1即倒数最后一个元素

tuple

与list不同，tuple一旦被确定就不能改变
