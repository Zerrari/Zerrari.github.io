Makefile

intro

编译文件的工具，可以有效组织大规模的源文件。

规则

command 前要用Tab键缩进

首先从依赖中寻找是否有比目标文件更新的文件，或者如果目标文件不存在，则运行下面的命令，并生成目标文件。

.PHONY表示clean是一个伪目标，并不会生成这个文件，只有显式执行make clean,才会执行命令。

变量

变量的声明
objects = program.o foo.o utils.o
变量的使用
program : $(objects)
