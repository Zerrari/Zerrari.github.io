安装重启前一定要安装好网络连接工具

解除rfkill的无线网卡禁用

rfkill list
如果soft blocked是no的状态
rfkill unblock wifi

tools

用pacman命令安装工具
sudo pacman -S dwm emacs st dmen networkmanager

  • dwm
    一个简洁的窗口管理器
  • emacs
    一个强大的代码编辑器s
  • st
    一个快速的终端模拟器
  • dmenu
    一个快速的程序启动器
  • networkmanager
    网络管理器
  • fish
    一个开箱即用的shell
