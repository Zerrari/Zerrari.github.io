效果

最后，通过，一系列快捷键就可以在Emacs中完成Hexo博客的相关操作

command function
M-p b start hexo
M-p RET open file
M-p SPC show article info
M-p D delete file
M-p N new file
M-p M a add tags
M-p M r remove tags
M-p m mark
M-p u unmark
M-p s r run server
M-p s s stop server
M-p s d deploy
